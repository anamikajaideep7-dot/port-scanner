#  Port Scanner

This is a simple Python-based port scanner that checks for open ports on a target system.

##  Features
- Scans ports from 1 to 1024
- Identifies open ports
- Uses Python socket programming

## How to Run
1. Install Python
2. Run: python port_scanner.py
3. Enter a website or IP

## 📚Learning Outcome
- Networking basics
- Port scanning concepts
- Cybersecurity fundamentals
